# KoreGlobe

## WARNING: Currently non-functional. Big rebuild. 

## Description

A 3D globe application, with assessible map and model data. 
Content can be added and animated through message and CLI interfaces. 

## Features
- Full 1:1 scale Earth, from orbit to 10m imagery and DEM. 
- Google Maps style UI movement. 
- Zoom in/out
- Pan view
- Load 3D models
- Adjust lighting
- Change camera angles
- Measure distances or angles
